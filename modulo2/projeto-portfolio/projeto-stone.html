Projeto de branding
e design web

O escopo do projeto, envole tanto a idealização da identidade visual para a StoneTech Máquinas, quanto a construção de um storytelling a fim de consolidar a jornada do usuário através das páginas. Para a construção do site foram utilizadas a plataforma Wordpress e o plugin Elementor. Para as peças gráficas, foi utilizado o Pacote Adobe, onde foram desenhados também cartôes de visitas e e-mail marketing.
