Os 7 passos fundamentais para resolver os problemas de programação estão relacionados a ter bastante atenção no que o enunciado está pedindo, saber separar as informações mais relevantes para a solução, pensar nessas informações dadas e requeridas como entradas e saídas dos programas, fazer algum tipo de mindmap pra ilustrar o código, criar um código funcional antes de pensar se ele está bonito ou não, testar a força bruta e verificar se o código está com algum gap, e por fim refatorar para deixar o código mais enxuto e adequado para o trabalho em equipe.
